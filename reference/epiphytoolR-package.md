# epiphytoolR: Epiphytotic Modeling Tools

epiphytoolR provides functions for undertaking epiphytotic modelling and
simulation studies in R. This package provides common base level
functions that can be used in modelling the spatial movement of splash
and wind dispersed plant disease inoculum.
